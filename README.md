# StudentManagementAPI
A modern Student Management API built with ASP.NET Core, featuring JWT authentication, secure REST APIs, database integration, and a professional frontend dashboard. Supports CRUD, search, and pagination fully deployable and perfect for learning full-stack development.

Features ✅

- REST API development with ASP.NET Core  
- JWT authentication & token-based authorization  
- Database integration with SQL Server  
- CRUD operations for students (Create, Read, Update, Delete)  
- Pagination and search functionality  
- Professional frontend dashboard  
- CORS support for frontend integration  
- Fully deployable application  
- GitHub repository ready for version control  

## Tech Stack

- **Backend:** ASP.NET Core, C#  
- **Database:** SQL Server, Entity Framework Core  
- **Authentication:** JWT (JSON Web Token)  
- **Frontend (Optional):** HTML, CSS, JavaScript
